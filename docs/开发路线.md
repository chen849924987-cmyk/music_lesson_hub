## Phase 4 - 学习体验

### 目标
提供优质的学习观看体验。

Phase 4 拆分为 **2 个子版本**。

---

### v4.1：视频播放器组件 + 学习进度追踪 ✅ 已完成

**目标：** 开发专用视频播放器组件，实现学习进度自动记录和断点续播。

#### 已完成功能
- [x] 创建 VideoPlayer.vue 专用播放器组件
  - 支持进度拖拽 + 缓冲进度显示
  - 播放速度调节（0.5x ~ 2x）
  - 画中画（PiP）模式
  - 全屏播放 + 键盘快捷键（空格/方向键/F/M）
  - 试看时长控制（requestAnimationFrame 高频监控）
  - 加载/错误/空状态展示
- [x] 学习进度追踪
  - 创建 LearningProgress entity（学习进度表）
  - 4 个 API 端点（更新/断点续播/课程进度/课时进度列表）
  - 自动记录播放进度（每 30 秒 + 暂停时同步后端）
  - 断点续播（进入课时自动恢复上次播放位置）
  - 学习完成标记（progress >= 95% 自动标记完成）
- [x] 学员端学习页面完善（CoursePlayer.vue 重构）
- [x] 课件下载功能（已购用户可下载附件，展开面板展示文件列表）

#### 涉及文件
- `frontend/src/components/VideoPlayer.vue`（新建 1008 行）
- `frontend/src/views/dashboard/CoursePlayer.vue`（重构 745 行）
- `frontend/src/api/learning.ts`（新建）
- `backend/src/modules/learning/`（新模块：Entity/Service/Controller/DTO/Module）

#### 验证标准
- ✅ 已购学员可流畅观看完整课程视频
- ✅ 视频播放器支持进度拖拽和速度调节
- ✅ 学习进度自动保存，刷新后可从断点续播
- ✅ 已购学员可下载课程附件

### v5.1：提现审核 + 日志清理 ✅ 已完成

**目标：** 实现教师提现管理端审核、系统日志自动清理。

#### 已完成功能
- [x] **提现全流程打通**
  - 创建 Withdrawal entity（提现申请表：金额/状态/收款账号/审核信息/处理时间）
  - 提现 DTO 校验（CreateWithdrawalDto + ReviewWithdrawalDto）
  - Service 层：教师申请提现（余额校验+冻结）、管理端审核（通过/驳回事务级处理）、
    提现记录查询（教师端/管理端）、待审核计数
  - Controller API：
    - `POST /earnings/withdrawals`（教师申请提现）
    - `GET /earnings/withdrawals`（教师提现记录）
    - `GET /earnings/admin/withdrawals`（管理端全部提现记录，支持状态筛选）
    - `POST /earnings/admin/withdrawals/:id/review`（审核通过/驳回）
    - `GET /earnings/admin/pending-count`（待审核数量）
  - 前端管理端提现审核页 AdminWithdrawals.vue
    - 状态筛选（待审核/已通过/已驳回/全部）
    - 审核通过/驳回操作（驳回必须填写原因）
    - 分页 + 骨架屏加载态 + 空状态
  - 教师端 TeacherEarnings.vue 提现表单已对接真实后端接口
  - 教师端 TeacherWithdrawals.vue 提现记录列表已对接真实数据
  - 路由注册：`/admin/withdrawals`
