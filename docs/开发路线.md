## Phase 4 - 学习体验

### 目标
提供优质的学习观看体验。

Phase 4 拆分为 **2 个子版本**。

---

### v4.1：视频播放器组件 + 学习进度追踪 ✅ 已完成

**目标：** 开发专用视频播放器组件，实现学习进度自动记录和断点续播。

#### 已完成功能
- [x] 创建 VideoPlayer.vue 专用播放器组件
  - 支持进度拖拽 + 缓冲进度显示
  - 播放速度调节（0.5x ~ 2x）
  - 画中画（PiP）模式
  - 全屏播放 + 键盘快捷键（空格/方向键/F/M）
  - 试看时长控制（requestAnimationFrame 高频监控）
  - 加载/错误/空状态展示
- [x] 学习进度追踪
  - 创建 LearningProgress entity（学习进度表）
  - 4 个 API 端点（更新/断点续播/课程进度/课时进度列表）
  - 自动记录播放进度（每 30 秒 + 暂停时同步后端）
  - 断点续播（进入课时自动恢复上次播放位置）
  - 学习完成标记（progress >= 95% 自动标记完成）
- [x] 学员端学习页面完善（CoursePlayer.vue 重构）
- [x] 课件下载功能（已购用户可下载附件，展开面板展示文件列表）

#### 涉及文件
- `frontend/src/components/VideoPlayer.vue`（新建 1008 行）
- `frontend/src/views/dashboard/CoursePlayer.vue`（重构 745 行）
- `frontend/src/api/learning.ts`（新建）
- `backend/src/modules/learning/`（新模块：Entity/Service/Controller/DTO/Module）

#### 验证标准
- ✅ 已购学员可流畅观看完整课程视频
- ✅ 视频播放器支持进度拖拽和速度调节
- ✅ 学习进度自动保存，刷新后可从断点续播
- ✅ 已购学员可下载课程附件

#### API端点
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/learning/progress` | 更新课时学习进度 |
| GET | `/learning/progress/:lessonId` | 获取断点续播位置 |
| GET | `/learning/course/:courseId` | 课程总进度统计 |
| GET | `/learning/course/:courseId/lessons` | 所有课时进度列表 |
