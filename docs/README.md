# 🎵 音乐视频课程销售平台

一个专注于音乐教育的视频课程在线销售平台，支持**单课程**和**系列课程**两种售卖模式，实现**管理端、教师端、制作人端**三权分立的后台管理体系。

---

## 📋 功能列表

### 🏢 管理端
- **仪表盘**：平台数据概览（总用户/总课程/总订单/总收益/热门课程排行及真实数据绑定）
- **用户管理**：用户列表（关键词搜索/角色筛选/分页）、启用/禁用、删除用户、创建账号（审核员/运营管理员）
- **教师管理**：创建教师账号、编辑信息、认证审核
- **课程审核**：课件上架审核、资料审核（含视频预览/附件审查）、下架管理
- **课程管理**：课程列表全量查看（状态筛选/推荐置顶设置）
- **订单管理**：订单查看、退款处理、报表导出
- **优惠券管理**：创建分发优惠券、使用统计
- **捆绑销售管理**：创建课程捆绑包、定价设置
- **提现审核**：教师提现申请审核与处理
- **数据统计**：销售报表、课程排行、收益统计
- **系统配置**：平台设置、支付/存储/提现配置

### 👨‍🏫 教师端
- **教师仪表盘**：课程数据统计、制作人概况、收益概览（收益趋势图）
- **课程管理**：创建课程（单课/系列课分离）、视频上传与管理、章节/课时编排
- **视频管理**：视频素材库（多文件管理/预览/删除）、上传进度条
- **附件管理**：课件/乐谱上传（拖拽上传/进度条）、审核状态跟踪
- **试看设置**：课程级默认试看时长 + 课时级覆盖（秒级精确控制）、视频试看播放接口
- **制作人管理**：查看已购课制作人列表（含学习进度查看）
- **收益管理**：收益概览（总收入/可提现/待结算/已提现统计）、收益趋势图（近30天折线图）、收益明细列表（分页+日期筛选）、课程销售排行
- **个人中心**：资料编辑、收款账号、通知偏好（统一纳入个人中心页面）

### 🏛️ 管理端 — 收益看板
- **平台收益总览**：总收入、平台分成、教师总收入、总订单数
- **收益趋势图**：近30天每日收益变化柱状图（总流水/平台收入/教师收益三色柱）
- **教师收益排行榜**：按总收益排列教师 TOP10
- **课程收益排行榜**：按总收益排列课程 TOP10
- **支持按日/周/月筛选查看**

### 🏠 公开首页（无需登录）
- **首页门户**：Hero 视频背景区、热门课程推荐（骨架屏加载态）、"关于我们"导师介绍（头像光环动画+技能标签+数据统计+社交链接）
- **课程中心**：独立公开页面 `/courses`，课程列表（分类导航、搜索、筛选、排序、分页加载）
- **课程详情**：公开课程详情页 `/courses/:id`，课程封面/简介展示、章节/课时列表、试看功能

### 🎛️ 制作人端（需登录，原"学员端"）
- **个人中心**：个人信息编辑（昵称/邮箱/手机号/简介）、头像更换、账号安全概览、角色专属设置区块（教师额外展示收款账号/通知偏好）
- **用户中心**：制作人仪表盘
- **我的课程**：已购课程列表，开始学习入口 `/producer/my-courses`
- **课程播放**：课程视频播放页 `/producer/courses/:id`，已购买学员完整播放，未购买学员限时试看
- **购课**：单课购买、系列课购买、系列课内课时单独购买、捆绑包购买（待 v4.x）
- **购物车**：课程加入购物车、批量结算
- **支付**：支付宝支付（扫码/跳转支付）、支付成功回调处理
- **我的订单**：订单查询、取消订单、申请退款
- **优惠券**：凭码领取优惠券、下单时选择可用优惠券抵扣
- **学习中心**：课程视频播放、学习进度追踪、课件下载（待 v4.x）
- **课程评价**：购买后可发表评价（评分+文字）（待 v4.x）
- **学习进度**：课程学习进度查看、续播（待 v4.x）

### 🔍 审核员端
- **审核工作台**：待审核课程列表（含课程详情/附件预览/视频试看）、审核通过/驳回弹窗
- **课程审核**：课程上架审核（含章节/课时内容/视频审查）、下架申请审核
- **附件审核**：独立附件审核页面（与课程审核分离），展示附件来源（单课程/系列课程/系列课时），支持通过/驳回操作
- **审核记录**：历史审核记录查询

### ⚙️ 运营员端
- **运营数据**：平台运营数据概览
- **订单管理**：订单查询和状态追踪
- **课程管理**：课程上架/下架管理、推荐置顶设置
- **优惠券管理**：优惠券分发与使用统计
- **捆绑包管理**：课程捆绑包创建与配置

## 🛠 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | Vue 3 + TypeScript + Vite | 前端框架 |
| **UI 主题引擎** | Vuestic UI (`--va-*` CSS 变量体系) | 双主题（亮色/暗色） |
| **UI 组件** | Element Plus | 后台 UI 组件覆盖主题 |
| **状态管理** | Pinia | 全局状态管理（含 theme store） |
| **路由** | Vue Router 4 | 前端路由 |
| **HTTP** | Axios | API 请求 |
| **后端** | NestJS + TypeScript | 后端框架 |
| **ORM** | TypeORM (Code-First) | 数据库自动建表、关系映射 |
| **数据库** | MySQL 8.0 | 关系型数据库 |
| **缓存** | Redis 7.x | 缓存/会话（可在 config 中配置） |
| **文件存储** | MinIO | 视频/附件对象存储 |
| **支付** | 支付宝（支付宝 SDK） | 在线支付 |
| **容器化** | Docker + Docker Compose | 部署运行 |

> 详细技术栈说明请参考 [`技术栈.md`](技术栈.md)

## 🚀 快速开始

### 环境要求
- Node.js >= 20
- npm / pnpm

### 1️⃣ 启动 MySQL 数据库

本项目使用你本机的 MySQL。如果尚未安装：
```bash
# 方式一：使用 Docker（推荐）
docker compose up -d

# 方式二：使用本地 MySQL（需要先安装 MySQL 8.0+）
```

### 2️⃣ 配置数据库与 MinIO 连接

编辑 `backend/.env` 中的连接信息（本项目默认使用**本机环境**，无需 Docker）：
```env
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=你的密码
DB_DATABASE=music_edu

# MinIO 配置（本机运行，默认账号密码为 minioadmin/minioadmin）
MINIO_ENDPOINT=127.0.0.1
MINIO_PORT=9000
MINIO_ACCESS_KEY=minioadmin
MINIO_SECRET_KEY=minioadmin
MINIO_BUCKET=music-edu
```
> ⚠️ **注意**：`docker-compose.yml` 仅供需要使用 Docker 替代本机环境的开发者参考，
> 本机开发请直接启动 MySQL 和 MinIO，无需执行 `docker-compose up`。
> 使用 Docker 启动 MinIO 时，需确保 `MINIO_ROOT_PASSWORD` 与 `.env` 中的 `MINIO_SECRET_KEY` 一致。

### 3️⃣ 创建数据库并启动后端

```bash
# 安装后端依赖
cd backend
npm install

# 创建数据库
node scripts/create-db.js

# 启动后端（开发模式）
npm run start:dev
```

后端启动后访问：**http://localhost:3000/api/docs**（Swagger API 文档）

### 4️⃣ 启动前端

```bash
# 新终端
cd frontend
npm install
npm run dev
```

前端启动后访问：**http://localhost:5173**

## 🌐 API 接口概览

所有 API 前缀为 `/api/v1`：

### 认证模块 — `/api/v1/auth`
| 方法 | 路径 | 说明 | 权限 |
|------|------|------|:----:|
| POST | `/auth/login` | 制作人登录 | 公开 |
| POST | `/auth/register` | 制作人注册 | 公开 |
| POST | `/auth/admin-login` | 管理端/教师登录 | 公开 |
| POST | `/auth/refresh` | 刷新 Token | 已登录 |
| POST | `/auth/logout` | 退出登录 | 已登录 |
| PUT | `/auth/change-password` | 修改密码 | 已登录 |

### 用户模块 — `/api/v1/users`
| 方法 | 路径 | 说明 | 权限 |
|------|------|------|:----:|
| GET | `/users/profile` | 查看个人信息 | 已登录 |
| PUT | `/users/profile` | 修改个人信息（含头像/手机号/通知偏好） | 已登录 |
| PUT | `/users/password` | 修改密码 | 已登录 |

### 教师模块 — `/api/v1/teachers`
| 方法 | 路径 | 说明 | 权限 |
|------|------|------|:----:|
| GET | `/teachers` | 教师列表 | 管理员 |
| POST | `/teachers` | 创建教师账号 | 管理员 |
| GET | `/teachers/:id` | 教师详情 | 管理员/教师 |
| PUT | `/teachers/:id` | 编辑教师信息 | 管理员 |
| PUT | `/teachers/settings` | 教师端更新设置（收款账号/通知偏好） | 教师 |
| PATCH | `/teachers/:id/status` | 启用/禁用教师 | 管理员 |
| PATCH | `/teachers/:id/verify` | 教师认证/取消认证 | 超级管理员 |

### 课程模块 — `/api/v1/courses`
| 方法 | 路径 | 说明 | 权限 |
|------|------|------|:----:|
| GET | `/courses/teacher` | 我的课程列表（教师端，含状态筛选） | 教师 |
