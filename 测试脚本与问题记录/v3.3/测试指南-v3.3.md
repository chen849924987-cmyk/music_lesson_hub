## 5. 常见问题排查

### Q1: 测试脚本无法连接后端
- 确认后端已启动：访问 http://localhost:3000/api/v1/courses
- 确认端口是否为 3000（如不同请修改脚本中的 BASE_URL）

### Q2: 重复码冲突测试失败
- 确认 TC-CPN-01 创建成功后再执行 TC-CPN-03
- 检查数据库是否已有同 code 的记录

### Q3: 优惠券计算返回 500
- 检查优惠券表结构是否完整（maxDiscount 字段可空）
- 检查后端日志查看具体错误堆栈

### Q4: 测试数据清理
- 测试脚本创建的优惠券码以 `cpn_` 开头，可在管理端手动删除
- 用户优惠券数据会自动入库
